## 競賽簡介
本競賽旨在開發能自動識別、分析和驗證企業永續承諾的 NLP 模型，提升企業 ESG 資訊揭露的透明度與可信度。使用 **VeriPromiseESG4K** 標註語料庫（4,000 筆繁體中文資料）完成四大核心任務。

## 四大核心任務

### 子任務一：承諾語句識別（權重 20%）
- **目標**：判斷文本是否包含企業對未來行動的明確承諾
- **評分指標**：F1-Score

### 子任務二：預期驗證時機推論（權重 15%）
- **目標**：推論承諾的預期完成時間
- **分類**：已實行 / 2年內 / 2-5年 / 5年以上 / N/A
- **評分指標**：Macro-F1 Score

### 子任務三：支持證據連結（權重 30%）
- **目標**：判斷承諾是否附有具體執行計畫或支持佐證
- **評分指標**：F1-Score

### 子任務四：證據清晰度評估（權重 35%）
- **目標**：評估證據語句的清晰度
- **分類**：Clear / Not Clear / Misleading / N/A
- **評分指標**：Macro-F1 Score

## 綜合評分公式
```
總分 = 承諾識別 F1 × 0.20 + 證據支持 F1 × 0.30
     + 清晰度 Macro-F1 × 0.35 + 時機預測 Macro-F1 × 0.15
```

## 資料集結構
每筆 JSON 資料包含以下欄位：
- `data`：原始文本段落
- `url`：永續報告書來源連結
- `page_number`：頁碼
- `esg_type`：E（環境）/ S（社會）/ G（治理）
- `promise_status`：Yes / No
- `promise_string`：承諾語句（若有）
- `verification_timeline`：already / within_2_years / between_2_and_5_years / more_than_5_years / N/A
- `evidence_status`：Yes / No / N/A
- `evidence_string`：證據語句（若有）
- `evidence_quality`：Clear / Not Clear / Misleading / N/A
